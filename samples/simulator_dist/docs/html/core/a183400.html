Detailed Description

pygameを用いて、画面の上側に戦域を上から見た図を、下側に戦域を南から見た図を描画する例。
機体、誘導弾、センサの覆域、防衛ラインの描画に加え、現在の時刻、得点、報酬、Agentの情報等を表示する。
