Detailed Description

仮想的な水平な慣性座標系を表すクラス

局所水平座標系に近いが、地球の自転と曲率を無視しており、慣性力や高度方向の歪みが無視される。

  • 基準座標系として設定した場合のみしか使用できない。
  • 座標軸は直交座標系のみとする。
  • 暗黙に何らかのGeodeticCRS上における原点情報を持つ。
