Detailed Description

EME2000やICRF等、地球中心かつ回転しない座標系。大抵の用途では慣性系とみなせる。

時刻を指定することで地球の回転状態を特定でき、ECEF等のGeodeticCRSに変換できる。 測地系ISO 19111の外にあるため、ECEFとの相互変換のみを定義し、それ以外とはECEFを介して変換するものとする。
