Detailed Description

Entity とその派生クラスのメンバに対するアクセス制限を実現するためのAccessorクラス。

pybind11に公開するメンバは原則publicとする必要があるため、protected相当のアクセス制限を実現することが難しい。 Accessorを介したアクセスに限定することで Entity インスタンス自身へのアクセスを防止し、 メンバごとのアクセス可否を細かく制御できるようにする。
