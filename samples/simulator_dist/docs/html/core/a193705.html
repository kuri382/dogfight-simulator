Detailed Description

航跡情報を表すための基底クラス

基底クラスでは座標系、時刻、PhysicalAsset識別子(truth)の情報のみを保持し、 実際には航跡の種類に応じてTrack3D又はTrack2Dを使い分けるものとする。
