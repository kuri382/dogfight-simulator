Detailed Description

複数の Agent インスタンスを組み合わせて一つの Agent として扱うための最もシンプルなクラス

ObservationとActionはchildrenのそれぞれの要素をdictに格納したものとする。

内部状態のシリアライゼーションを行う。 (Experimental)

Parameters
[in,out]archive入出力対象のArchive
[in]full全状態量を入出力するかどうか。falseの場合、時間変化しない定数等は省略される。ただし、実際に何が入出力されるかは Entity 及び派生クラスでの実装依存。
