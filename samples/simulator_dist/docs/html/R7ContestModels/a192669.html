Detailed Description

R7年度コンテスト用の簡略化した機体運動モデルを用いる戦闘機クラス

迎角と横滑り角を無視し、機体のX軸と速度方向が常に一致するものとして速度ベクトルを直接操作する。 速度、加速度、角速度(速度ベクトルの変化率として)を高度依存のテーブルデータとして与える。
