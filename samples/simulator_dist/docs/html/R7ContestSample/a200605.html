Detailed Description

入力を複数のブロックに供給してその結果を結合するブロック。configは以下の形式とする。
{
    "input_shape": list, #入力テンソルの形(バッチサイズを除く)
    "blocks": list[dict], #分岐・結合対象の各ブロック(GenericLayers)を生成するためのconfig。
    "dim": int, #結合対象の次元。省略時は-1(末尾)となる。

}
